DETECTION AND CLASSIFICATION OF ARRHYTHMIA

A machine learning–based system to automatically detect and classify cardiac arrhythmias from ECG signals, enabling faster diagnosis and continuous monitoring for patients at risk.

Manual Input Upload CSV Analyze ECG Image

About Arrhythmia

Arrhythmia is an irregular heartbeat that disrupts normal heart function, often causing palpitations, dizziness, or fatigue. Diagnosis is typically done through electrocardiogram (ECG) signals, and treatment ranges from lifestyle changes and medication to implantable devices or surgery.

Algorithm

We use a Weighted KNN (k-Nearest Neighbors) model for classification. Each neighbour contributes to the prediction based on its distance from the query point, improving robustness and handling class overlap more effectively in ECG feature space.

Efficiency

Using carefully selected ECG features and Weighted KNN, our model reaches an accuracy of 98% on the test set. This shows that machine learning can effectively support cardiologists in early and reliable arrhythmia detection.

About the ECG Data

We use the Cardiac Arrhythmia dataset from Kaggle, containing around 1356 records across multiple arrhythmia classes. The pipeline includes preprocessing, feature extraction and scaling, followed by training and evaluation of the classifier.

How the System Works

The project entails creating an web-based system for real-time cardiac arrhythmia detection using ML algorithms. It involves developing compact to monitor cardiac parameters like heart rate and ECG intervals. Algorithms and machine learning models are employed for real-time analysis, accurately identifying arrhythmias. A web-based dashboard provides healthcare professionals and patients with visualizations of vital signs, alerts for detected arrhythmias, and historical data. The system is designed to be scalable, reliable, and user-friendly, with integration capabilities for existing healthcare systems. Testing, validation, documentation, and training are crucial steps for successful deployment in clinical settings.
